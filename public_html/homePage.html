Application Overview

Query Page

This Application Can Be Used To Perform Two Different Types Of Queries.

Customer Query

  • In this query, the "Customers" table is used from the database
  • The query displays customer information such as name, city, state, and country
  • The information is shown in ascending order by customer name
  • User input is the country name such as "USA"

Order Query

  • In this query, the "Orders" table is used from the database
  • The query displays the total order number based on the status
  • User can select any radio button from the list, and the query will show the results
